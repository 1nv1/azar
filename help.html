AZAR

It's a very simple game. The goal is guess the largest number from ten turns and select it.

The game starts when the player A or B click on 'Generate'. The other player see a random number from 0.0 to 10.0. Then that player has the option to select that number or not, the goal is try to select the largest number that you opponent generate. You can select a number only once. If one or both players has selected a number the game automatically generate the rest of the numbers.

Who wins? When the turns finish, the game check if you are select the largest number of the list generated by you opponent and give to you a point. If you and you opponente has 0 or 1 point is a draw but if only one have a point, this player wins the round.
